Why JavaScript is so popular in web development

It is easy, fast, and powerful. It doesn't require access to a server, compiling programs, or having the browser load to a virtual machine. All you need to do is add it to your webpage and upload the page. Running it is as simple as reloading the page. It gives you access to many features of the web page, browser and computer. It features an object-oriented achitecture like Java and C++. BUT IT IS NOT THE SAME A JAVA. It has a reduced server load, allowing for negotiations to be handled on the client side, leaving the server free for more important things.
